# Panzzoni web completo

Este ZIP está preparado para subirlo directamente a tu repositorio.

## Qué incluye
- `index.html` landing principal
- `gijon.html` página de Gijón
- `santiago.html` página de Santiago / Los Tilos / Pathos
- `menu.html` menú visual
- `galeria.html` galería
- `assets/images` fotos de pizzas y local
- `assets/docs` PDFs originales de carta y manual
- `assets/css/styles.css`
- `assets/js/site.js`

## Cómo publicarlo
1. Descomprime este ZIP.
2. Arrastra todos los archivos dentro de tu repo de GitHub.
3. Haz commit.
4. Vercel lo publicará automáticamente.

## Nota
Si tu repo aún tenía archivos antiguos, sustitúyelos por esta versión.
