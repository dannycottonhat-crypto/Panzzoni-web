# PANZZONI

Tradizione italiana. Forno a legna.

Panzzoni nace para traer a la mesa una forma auténtica de entender la cocina italiana: 
masa madre de larga fermentación, ingredientes italianos de alta calidad y respeto 
por los tiempos tradicionales.

La web mostrará:

- Historia del proyecto
- Establecimientos
- Carta
- Fotos
- Acceso a reservas y pedidos

Tecnología:

Next.js
Vercel
GitHub
